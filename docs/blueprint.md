# **App Name**: VisionQuest

## Core Features:

- Multiplayer Game Modes: Engage in three distinct game formats: Solo Arena for training, 1v1 Duel Arena for direct competition, and a 50-player Battle Royale mode.
- Semantic AI Consensus: Utilize GenLayer AI as a tool to evaluate user submissions based on semantic meaning rather than literal text matching, ensuring fair and creative judging.
- Dynamic Image Matrix: Interactive 2x2 image grids featuring rounded corners, subtle glows, and responsive layout for competitive puzzle solving.
- Real-time Arena Lobby: A pre-game coordination screen displaying player avatars, wallet badges, current ranks, and an animated match countdown.
- Cinematic Ledger Processing: An immersive on-chain loading sequence featuring a pulsating GenLayer orb and status message rotations during consensus formation.
- Global Performance Podium: Visual completion screens with rank adjustments, XP earned, and a winner's podium for top placements in competitive rounds.
- On-Chain Reward Distribution: Direct interaction with the GenLayer protocol to finalize prize pools and update state records securely across the network.

## Style Guidelines:

- Primary Color: Electric Violet (#9D50FF) symbolizing futuristic energy. Background Color: Deep Obsidian (#0E0B16), a heavily desaturated variant for a dark cyberpunk arena feel. Accent Color: Radiant Cerulean (#508BFF), analogous to the primary for vibrant interactive highlights.
- Headline font: 'Space Grotesk' (Sans-serif) for its high-tech, geometric vibe. Body font: 'Inter' (Sans-serif) for clean readability and neutral gaming hierarchy.
- Neon-style iconography with subtle golden outlines to distinguish competitive rewards and prize pool items.
- Premium mobile-first design utilizing centered card stacks for game modes and a responsive 2x2 grid for main puzzle interactions.
- Dynamic page transitions and Framer Motion powered orb pulses during AI consensus, ensuring the UI feels alive rather than static.
